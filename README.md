# Omni-ChiselStore

Omni-ChiselStore is an embeddable, distributed [SQLite](https://www.sqlite.org/index.html) for Rust, powered by [Omni-Paxos](https://github.com/haraldng/omnipaxos).

## Run tests
Build tests
```bash
cargo build --tests
```
Run a test
```bash
cargo test test_consistency_relaxed
```
